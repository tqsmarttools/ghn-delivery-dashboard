# GHN Delivery Dashboard

Public mobile PWA shell for Thien Quang GHN failed-delivery workflow.

This repository is intentionally public so GitHub Pages can host it for free.

Safety rules:

- No GHN API token.
- No real customer data.
- No SQLite database.
- No workflow scripts.
- Sample orders use fake phone numbers only.

The private workflow repository remains separate and is used to scan GHN and prepare sanitized dashboard data.
